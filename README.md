# Runner-Progress-Tracker

This application tracks a runner's progress in a marathon and calculates the speed required to reach 
the finish line within a specified target time. 

The user can input the following details:

The total marathon distance (50km).

The distance the runner has already covered.

The elapsed time since the start.

The target time to complete the marathon.




< --- Additionally, the application can --- >

Calculate the runner's current average speed.

Determine the speed required to finish within the target time.

Store historical race data in a multidimensional array.

Format results for clear presentation.

Save calculation data to a file for future analysis.

Use functions for calculations and data manipulation.
